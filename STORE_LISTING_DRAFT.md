# Store Listing Draft

## Extension Name

NotebookLM Slide Command Helper

## Short Description (45 chars max)

Generate & inject slide-edit prompts into NotebookLM with AI.

## Detailed Description

### English

NotebookLM Slide Command Helper helps you edit slides faster by turning your intent into high-quality editing prompts.

**Key Features:**

🎨 **218 Design DNA Templates** — 10 core + 208 Darlkom Banana design presets across 7 categories
🔧 **Custom Prompt Builder** — Combine goal, target, tone, length, and extra constraints
🤖 **Gemini Vision AI Analysis** — Screenshot-based slide diagnosis with 6-Gene DNA framework and fitness scoring
💾 **Save & Reuse** — Store your best prompts and access recent history
⚡ **One-click Inject** — Send prompts directly to NotebookLM slide edit inputs
🌙 **Dark Mode** — System-aware theme with manual toggle
⌨️ **Keyboard Shortcut** — Ctrl+Enter to inject instantly

**AI Quality Controls:**
- Slide type mode (auto/business/pitch/education/data)
- Improvement level (conservative/balanced/aggressive)
- Review language (Korean/English/Bilingual)

Designed for presenters, educators, and teams who iterate slide content frequently.

---

### 한국어

NotebookLM 슬라이드 커맨드 헬퍼는 슬라이드 수정 의도를 고품질 프롬프트로 변환해 빠르게 편집할 수 있도록 도와줍니다.

**주요 기능:**

🎨 **218개 디자인 DNA 템플릿** — 핵심 10개 + Darlkom Banana 208개 프리셋 (7개 카테고리)
🔧 **맞춤 프롬프트 빌더** — 목표, 대상, 톤, 길이, 추가 조건을 조합
🤖 **Gemini Vision AI 분석** — 스크린샷 기반 슬라이드 진단 + 6유전자 DNA 프레임워크
💾 **저장 & 재사용** — 자주 쓰는 프롬프트를 저장하고 최근 기록 관리
⚡ **원클릭 주입** — NotebookLM 슬라이드 편집 입력창에 바로 전송
🌙 **다크 모드** — 시스템 설정 연동 + 수동 전환
⌨️ **단축키** — Ctrl+Enter로 즉시 주입

## Privacy Highlights

- No tracking
- No third-party data sharing (except user-initiated Gemini API calls)
- Local-only storage for all saved data
- User's own API key required for AI features
